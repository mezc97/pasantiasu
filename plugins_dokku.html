Instalando Plugins

Dokku permite instalar plugins para conectarlos a nuestras aplicaciones, tales como base de datos, servicios, etc. En base a los servicios que use nuestro proyecto o app debemos instalar los plugins adecuados para que funcione correctamente. Entre lo más conocidos tenemos MySQL, Postgres, Elasticsearch, Redis, etc
Para instalar plugins consultamos las instrucciones para cada plugin en específico. Por ejemplo visitamos el plugin de MySQL, leemos sus instrucciones de instalación en la cual debemos ejecutar con permisos root el siguiente comando:
sudo dokku plugin:install https://github.com/dokku/dokku-mysql.git mysql
Dokku “plugins” igualmente es un módulo de dokku donde podemos ejecutar diferentes acciones a nuestros plugins.
Muestra una lista de los plugins activos
dokku plugin
Desinstala un plugin
dokku plugin:uninstall postgres
Deshabilita un plugin instalado
dokku plugin:disable postgres
Habilita un plugin previamente deshabilitado
dokku plugin:enable postgres
Actualiza el plugin desde su repositorio
dokku plugin:update postgres
