Almacenamiento Persistente

Dentro de la aplicación existen rutas o carpetas donde se almacenan imágenes y otros componentes que a veces el desarrollador desea que sean guardadas o permanezcan dentro de las rutas del proyecto. Para lograr que estas carpetas y archivos permanezcan siempre debemos decirle a dokku que mapee o establezca una ruta para guardar todo lo que se suba a dicha ruta del proyecto.
Primero creamos un directorio de almacenamiento en el servidor host
mkdir -p /storage/app/public/uploads
Luego le decimos a dokku que guardar los archivos en el directorio que acabas de crear
dokku docker-options:add appname deploy, run -v /storage/<appname>/<environment>/public/uploads:/app/public/uploads”
Para finalizar debemos reconstruir la app para que tome los cambios de almacenamientos persistente
dokku ps:rebuild <app-name>
