Configurando y Enlazando Plugins

Para enlazar los plugins a nuestra app debemos crear un contenedor y posteriormente enlazarlo a nuestra aplicación. Usaremos como ejemplo el plugin de MySQL para enlazar un servicio de base de datos a una aplicación.
dokku mysql:create <name>
Nota: por defecto el plugin instala la última versión estable para cada plugin, en el caso de MySQL instala la version 5.7.12.
Si queremos crear un contenedor de MySQL con una versión en específica, antes de crear el contenedor debes ejecutar los siguientes comandos:
export MYSQL_IMAGE="mysql"
export MYSQL_IMAGE_VERSION="5.5"
dokku mysql:create stagingDB
Estos comandos crean un contenedor o servicio de mysql con la versión 5.5. Cualquier version no es compatible, debes verificar la documentación en su repositorio oficial. Este mismo método de instalación aplica para diferentes plugins como elasticsearch, postgres, entre otros.
Una vez tengamos nuestro contenedor creado debemos enlazarlo a una aplicación. Para ello debemos ejecutar el siguiente comando:
dokku mysql:link <name> <app-name>
ejemplo:
Dokku mysql:link stagingDB staging
De esta manera enlazaremos nuestro servicio o contenedor de mysql llamado “stagingDB” a la aplicación llamada “staging”, adicionalmente creará una variable de entorno llamada “DATABASE_URL” en la cual debes insertarla en el código de tu aplicación.
