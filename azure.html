Azure

SERVIDORES CLOUD

servidores vituales privados.

1. Crea una cuenta e inicie sesión con Azure
Ingresa en la pagina https://azure.microsoft.com/es-es/ y registrese o inicie sesion
2. Hacer clic en crear un recurso
En el menu que se genera selecionar App Web
3. Configuraciones
Establece :
a) Nombre de la app web
b) Suscriptor(Predeterminado)
c) Selecciona Sistema Operativo Windows - Linux - Docker
4. Seleccionar Plan
Se puede seleccionar uno que se ha usado anteriomente
Se puede Crear un nuevo plan que se adapte a las necesidades
5. Crear!
hacemos clic en crear
6. Configuración de la conexión
Clic en deployment option
Seleccionamos la fuente : Dropbox, GitHub, entre otros .
Clic en deployment credencial
colocamos usuario y password
7. Conectarse a git(si se ha seleccionado este metodo)
1. Accedemos al CMD
2. Ubicamos la direccion de la carpeta local del proyecto para crear repositorio
3. En la CMD usamos el comando git init
4. Para agregar el repositorio git add
5. Creamos el commit git commit –m “nombreDelCommit”
8. Conectar Pagina con Azure
a) Para comprobar que ya tenemos el remoto colocamos git remote –v
b) Para hacer push git push origin master(es master al ser la primera rama)
