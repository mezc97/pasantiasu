Implementar un Contenedor de Muestra Preconstruido
1. Inicie sesión en su cuenta de Google Cloud
Crear una cuenta en https://cloud.google.com/
2. Seleccione o cree un proyecto GCP
Tiene la opcion de crear o utilizar un proyecto anteriormente desarrollado
3. Asegúrese de que la facturación esté habilitada para su proyecto de Google Cloud Platform .
Aprenda a habilitar la facturación
4. Despliegue del contenedor de muestra
4.1) Dirigete a cloud run
4.2) Haga clic en Crear servicio para mostrar el formulario Crear servicio .
En la forma
a- Úselo gcr.io/cloudrun/hellocomo una imagen de contenedor.
b- Seleccione Cloud Run (totalmente administrado) como su plataforma de desarrollo.
c- Seleccione la región donde desea ubicar su servicio.
d- Seleccione Permitir invocaciones no autenticadas para poder abrir el resultado en su navegador web
e- Haga clic en Crear para implementar la imagen en Cloud Run y ​​espere a que finalice la implementación.
f- Haga clic en el enlace URL que se muestra para ejecutar el contenedor desplegado.
