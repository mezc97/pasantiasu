Obtener Root Password de los contenedores de servicios

Obtener Root Password de los contenedores de servicios: Dokku mantiene su propia estructura de archivos para organizar los servicios que utiliza y almacena ciertos datos que son muy útiles para el administrador, tales como el Root Password y archivos de configuración. La estructura de archivos es la siguiente:
/var/lib/dokku/services/mysql/your-service/ROOTPASSWORD or PASSWORD
