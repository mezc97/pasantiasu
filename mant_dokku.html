Mantenimiento a los servicios dokku

Al instalar una base de datos. Después del primer despliegue, lo más probable es que fracasen. Esto puede arreglarse fácilmente si ha instalado un complemento de base de datos en un paso apropiado. Para MySQL o Postgres:
dokku run staging rake db:create db:migrate
Para visualizar las aplicaciones
ssh dokku@server-IP apps:list
