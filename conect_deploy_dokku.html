Conectandonos a dokku para hacer Deploy

Dokku nos permite ejecutar comandos y realizar funciones para administrar completamente la aplicación de manera remota desde los ordenadores de los desarrolladores dándoles la facilidad con simples comandos desplegar la app en poco tiempo.
1. Debemos agregar nuestro ssh-key al host servidor para obtener permisos de ejecutar comandos dokku
cat ~/.ssh/authorized | ssh root@server-IP "sudo sshcommand acl-add dokku $USER"
o
cat ~/.ssh/id_rsa.pub | ssh root@server-IP "sudo sshcommand acl-add dokku $USER"
2. Desde tu ordenador local o pc del desarrollador, clonas tu proyecto en un directorio y navega hacia el:

git clone https://github.com/heroku/node-js-sample

cd node-js-sample

3. Agregamos el repositorio remoto para hacer despliegue en el servidor dokku:
git remote add <app-name> dokku@server-IP:<app-name>
4. Realiza un Push de rama master local a Dokku:
git push <app-name> master
También puedes realizar push desde otra rama local a dokku:
git push <app-name> <rama-local>:master
