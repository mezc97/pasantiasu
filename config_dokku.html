Configurando Nombres de Dominio

En dokku podemos asignar uno o varios dominios a nuestras apps. El servidor mapeara el dominio a una app que tu le indiques. Para asignar un dominio a una aplicación usa el siguiente comando:
dokku domains:add <app> <domain>
“Domains” es un módulo de dokku donde podemos realizar administrar otras funciones de los dominios. Para más Info consulta su documentación aquí. Algunas de las diferentes funciones que podemos realizar se encuentran en los siguientes comandos:
Muestra una lista de los dominios
dokku domains:add <app> <domain>
Añade un dominio a una app
domains:add <app> <domain> [<domain> ...]
Añade un dominio global
domains:add-global <domain> [<domain> ...]
Elimina todos los dominio de una app
domains:clear <app>
Deshabilita el soporte de VHOST
domains:disable <app>
Habilita el soporte de VHOST
domains:enable <app>
Remueve un dominio de una app
domains:remove <app> <domain> [<domain> ...]
Remueve un dominio global
domains:remove-global < domain> [< domain> ...]
Establece dominio para una app
ddomains:set <app> <domain> [<domain> ...] ...]
Establece dominio global para una app
domains:set-global <domain> [<domain> ...]
