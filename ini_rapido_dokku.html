Desplegar rápido una app en dokku

Pasos para Desplegar rápido una app en dokku:

1. Creamos la app con el siguiente comando
dokku apps:create staging
2. Creamos los contenedores o servicios que usara nuestra app
Dokku mysql:create stagingDB
3. Establecemos las variables de entorno para nuestra app
Dokku config:set staging USERNAME=admin PASSWORD=1234
4. Enlazaremos nuestros servicios a nuestra app
Dokku mysql:link stagingDB staging
5. Establecemos el dominio para la app
Dokku domain:add staging www.midominio.com dominio.com.ve
6. Configuramos el almacenamiento persistente en caso de nuestra app necesite almacenar archivos dentro de los directorios del proyecto
Creamos un directorio de almacenamiento en el servidor host dokk
Luego le decimos a dokku que guardar los archivos en el directorio que acabas de crear:
dokku docker-options:add appname run "-v /storage/<appname>/<environment>/public/uploads:/app/public/uploads”
Para finalizar debemos reconstruir la app para que tome los cambios de almacenamientos persistente:
dokku ps:rebuild <app-name>
7. Nos conectamos remotamente desde el ordenador para realizar deploy a nuestra app
Agregamos el repositorio remoto para hacer despliegue en el servidor dokku:
git remote add <app-name> dokku@server-IP:<app-name>
Realiza un Push de rama master local a Dokku:
git push <app-name> master
También puedes realizar push desde otra rama local a dokku
git push <app-name> <rama-local>:master
